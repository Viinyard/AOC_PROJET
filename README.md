## Maven

### Compiler le projet
  ```bash
  mvn compile
  ```

### Executer les tests
  ```bash
  mvn test
  ```

### Générer le jar
  ```bash
  mvn package
  ```
### Executer le jar
  ```bash
  mvn java -jar target/m3-1.0-SNAPSHOT.jar
  ```
